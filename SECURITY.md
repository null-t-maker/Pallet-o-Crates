# Security Policy

Thanks for helping keep Pallet-o-Crates safe to use.

## Supported Versions

This project currently ships as experimental `0.x` releases.

Security fixes, when needed, are handled on a best-effort basis in the latest public release line.

| Version | Supported |
| --- | --- |
| Latest `0.x` release | Yes |
| Older prereleases | No |

## Reporting a Vulnerability

Please avoid posting sensitive vulnerability details in a public issue.

Preferred path:

1. Use GitHub private vulnerability reporting for this repository if it is available.
2. If private reporting is not available, open a normal GitHub issue with only a high-level description and clearly mark it as a security concern.
3. Wait for maintainer follow-up before sharing exploit details, credentials, or proof-of-concept payloads publicly.

Please include:

- affected version or tag
- affected platform
- reproduction steps
- impact
- any suggested mitigation or workaround

## Scope

Reports related to malicious dependencies, unsafe file handling, exposed credentials, release integrity, or code execution risk are in scope.
